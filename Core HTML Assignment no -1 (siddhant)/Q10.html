1. HTML

    2. Primary Use Case Key Feature/Benefits
    HTML is used for structuring the content of web pages.
    • Semantics: HTML provides a wide range of semantic tags that help describe the meaning and structure of content, making it accessible and SEO-friendly.
    • Nesting: HTML allows elements to be nested within one another, enabling hierarchical organization of content.
    • Cross-Browser Compatibility: HTML is supported by all major web browsers, ensuring that web pages can be viewed consistently across different platforms.

  2. CSS

    3. Primary Use Case Key Feature/Benefits
    CSS is used for styling and visually formatting web pages.
    • Selectors: CSS allows the selection of specific HTML elements or groups of elements to apply styles.
    • Cascading: CSS follows a cascading model, where styles can be defined at different levels (inline, internal, or external stylesheets) and are applied based on specificity and inheritance rules.
    • Responsive Design: CSS provides features like media queries and flexbox/grid layouts, enabling responsive designs that adapt to different screen sizes and devices.

  3. JS

    4. Primary Use Case Key Feature/Benefits
    JS is a programming language that adds interactivity and dynamic behavior to web pages.
    • Manipulating HTML/CSS: JavaScript can access and modify the structure and style of HTML elements and CSS properties dynamically.
    • Event Handling: JS allows the handling of user interactions (e.g., clicks, keypresses) and executing corresponding actions or functions.
    • Asynchronous Operations: JavaScript supports asynchronous programming, allowing the execution of non-blocking operations like AJAX requests, timers, and animations.
